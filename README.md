# Config

## Sync git config

To sync the ***local repository*** and the ***.gitconfig*** use:

- **Unix/Linux:** `ln ./git-config/.gitconfig .gitconfig`
- **Windows:** `mklink .gitconfig ./git-config/.gitconfig`
